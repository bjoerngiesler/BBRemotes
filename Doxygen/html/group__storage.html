Detailed Description

To store within non-volatile memory, protocols can serialize themselves into structures of type StorageBlock. These capture all relevant info – paired nodes, receiver mixes, etc. The library does not implement the actual storage process into NVM, because that is highly dependent on the microcontroller and its NVM support library.

You typically do not want to call these functions directly, rather rely on ProtocolFactory for that.
